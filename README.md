# LastShift

Unreal Engine 5.6 project for **LastShift**.

## Project Layout

- `Source/` contains the C++ gameplay module plus horror and shooter variants.
- `Config/` contains project and editor defaults.
- `Content/` contains maps, blueprints, audio, characters, and environment art packs.

## Repository Notes

- Generated Unreal folders are intentionally ignored.
- Large binary assets are tracked with Git LFS.
